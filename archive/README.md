# Archive

This folder contains historical and portfolio projects that predate the active analytics engine stack.

These projects are **not part of the active workflow** (`intake_engine`, `metrics_engine`, `report_engine`).
They are preserved here for portfolio reference and prior learning.

## Contents

| Folder | Description |
|---|---|
| `data_center_agent/` | Experimental market agent script — early exploration, not part of engine stack |
| `excel/` | Excel-based data cleaning and analysis projects |
| `sql/` | SQL schema, analysis, and capstone projects |

## Rules

- Do not modify files in `archive/` unless explicitly requested by the user.
- Do not reference `archive/` from engine source code.
- Do not add new work to `archive/` — active work belongs in `intake_engine/`, `metrics_engine/`, or `report_engine/`.
