# Financial Data Analytics

## Objective
Analyze financial and market data to extract insights using SQL, Python, and Power BI.

## Tools Used
- PostgreSQL / SQL
- Python (pandas, matplotlib)
- Power BI
- Excel

## Project Structure
- `sql/`: SQL queries and schema
- `data/`: financial datasets (processed & samples)
- `notebooks/`: exploratory and analytical notebooks
- `powerbi/`: dashboards and visualizations
